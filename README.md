# My Cypress Automation

This project utilizes Cypress to automate the testing of a login feature and a user update feature.
Cypres is a Javascript-based end-to-end testing framework that allows developers to write tests and execute them in a real browser. 


## Installation
```bash
npm install
```

> **NOTE:**
>
> It is necessary to have **Node.js** installed

## Running
```bash
# Open Cypress UI
npx cypress open 

# Run tests in headless mode
npx cypress run
```
